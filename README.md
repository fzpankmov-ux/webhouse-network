# WebHouse Network

Red social 3D descentralizada — cada persona tiene su casa en un mundo navegable.

## Setup

```bash
npm install
npm run dev
```

Abre http://localhost:3000

## Controles

| Tecla | Acción |
|-------|--------|
| WASD | Manejar |
| Shift | Boost |
| Space | Freno |
| E | Entrar a casa |
| Escape | Cerrar panel |
| Mouse | Orbitar cámara |
| Scroll | Zoom |
